## Галерея для сайта

**Плагин для WordPress, который создает галерею для сайта шорткодом**

````
[gallery ids='1,2,3']
````
Где значение поля `ids` это `id` нужной вам картинки.
````bash
$ git clone https://github.com/stylesam/slsm_gallery.git
````
